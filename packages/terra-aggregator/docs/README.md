# Terra Aggregator

## Getting Started

- Install with [npmjs](https://www.npmjs.com):
  - `npm install terra-aggregator`
  - `yarn add terra-aggregator`

## Usage

```jsx
import React from 'react';
import Aggregator from 'terra-aggregator';

<Aggregator>
  ...
</Aggregator>
```
